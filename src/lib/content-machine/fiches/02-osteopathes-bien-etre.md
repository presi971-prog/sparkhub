# Ostéopathes & praticiens bien-être — Guadeloupe

## 1. Les 5 vraies galères
1. **Mains dans le dos d'un patient quand le téléphone sonne.** Une séance ne s'interrompt pas.
2. **L'appelant tombe sur le répondeur et raccroche.** > 60% raccrochent sans message (Servitel, FR). Quand on a mal, on veut un RDV maintenant.
3. **Le "je vous rappelle" non tenu.** Le patient a déjà pris ailleurs.
4. **Pas de secrétaire, pas les moyens (2 200 €/mois).** Cabinets GP = mobile 0690 + Gmail, praticiens seuls.
5. **Les lapins (no-show).** ~3-4% toutes spés (Doctolib 2024), créneau ostéo perdu = ~50-70 € + place gâchée. Sans rappel auto, le créneau saute.
